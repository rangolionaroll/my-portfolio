Tumultuous Absence: Virtual Gallery

This project is a virtual worlds experience designed for the 26th annual New Jersey Book Arts Symposium in November 2020.

Roles: 3D Artist, Designer, Developer, Marketing Coordinator

Design & Development


This project was designed to overcome some of the challenges of gathering due to COVID-19. The New Jersey Book Arts Symposium would generally host an exhibit during their event for artists to mingle and view each other's artworks. While they were able to shift their workshops to a virtual format over Zoom, the challenge was to create an environment for the community to be able to freely roam and talk.

For the technology behind this project, we used Mozilla Hubs which is a browser based virtual world, which lets users create and publish their own virtual worlds.

    The most important thing when designing this virtual world was to make it reflect the real life experience of being at an art gallery.

  • Showcasing the artists
    Many of the artist's works were sculptural. To showcase this aspect of the art I created 3D models using images of the artworks. The gallery layout was also designed to give each artist room to showcase their work and bio.

  • Immersing the visitors To make the gallery feel real, we included a lot of things you would see in real gallery including wall decals and a map. To add to the celebration feel, we added a refreshments table and a guestbook, users could stop by.
