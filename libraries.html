Virtual Rutgers Library

This project is a self-guided virtual worlds experience designed for the Rutgers University Libraries as an orientation space for new students.

Roles: 3D Artist, Designer, Developer

Design & Development

For the technology behind this project, we used Mozilla Hubs which is a browser based virtual world, which lets users create and publish their own virtual worlds. To showcase the services that are available at the Rutgers Libraries, we created 10 virtual rooms, each showcasing a room in the Mabel Smith Douglass Library on the Rutgers New Brunswick Campus.

    I laid out the world so that each room would present the same sequence of information as shown below:

    • The user would start at a Quickstart guide and signpost so that they would know which room they were in and the controls to move around.
    • Then they would move forward and see the 3D model, which was large enough to get close to and walk around.
    • They would then be able to play a video describing the services related to that room. The user can listen while they continue looking around or they could read the transcription on the screen.
    • Then finally they would come up to the scene select panel which would allow them to explore the other rooms of the library.
